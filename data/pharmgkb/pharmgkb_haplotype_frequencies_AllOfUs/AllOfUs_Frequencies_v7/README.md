# AllOfUs Frequencies from PharmGKB

All of Us Research Program frequencies were estimated from whole genome sequencing data on
the v7 cohort (n=245k) using [PharmCAT v2.4.0][5]. For more information on the methods and
quality control please see the [featured workspace][1] in the All of Us Researcher Workbench
(login required) or the [PGx frequencies article][2]. For more information on the dataset see
the [genomics article][3] and the [Genomics Quality Report][4].

For more information, read about [biobank frequencies available through PharmGKB][6].
